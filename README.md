# shokika.css

A lightly modern reset CSS.

## Installation

**npm**
```
npm install --save-dev shokika.css
```

**Yarn**
```
yarn add shokika.css
```

**CDN**
```html
<link rel="stylesheet" href="https://unpkg.com/shokika.css@1.0.0/shokika.min.css" />
```

## Usage

```css
@import 'node_modules/shokika.css/shokika.min.css';
```

or 

```html
<link rel="stylesheet" href="https://unpkg.com/shokika.css@1.0.0/shokika.min.css" />
```
